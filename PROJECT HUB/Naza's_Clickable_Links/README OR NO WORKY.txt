How To setup Clickable Links
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------


Step 1. (needed to host code to link to) (Skip if already created and login)
-Make a GitHub account
Note you can reference the file locally and it works without linking to gitbub, but it doesn't work on a server without the mod installed to all clients

Step 2.
-Make a new Public Repository. ( probably a way to do it private with tokens if you really wanted )

Step 3.
Open the "links.xml" file (you can rename if you want) found in the same directory as this readme \Naza's_Clickable_Links
add your link to the correct spot in that file and save.

Step 4.
Upload the "links.xml" file (or whatever you called it if renamed) to your GitHub repo

Step 5.
click on the file on GitHub and on the right you should see a "raw" button and click it
Copy the url from the address bar and go back to the mod.

Step 6.
Navigate to and open the windows.xml file found \Naza's_Clickable_Links\Config\XUi
Line 12 look for "sources" and paste your GitHub link there and save. Should look something like

https://raw.githubusercontent.com/SOMEUSER/REPO/main/SOMENAME.xml

Step 7.
dump the mod into your mods folder in appdata

%appdata%/\7DaysToDie\Mods (windows)

Step 8. You can make as many buttons as you want just duplicate everything for this button, rename the windows and upload new link.xml files to github etc feel free to resize, move, recolour etc whatever you wish to suit your menu.
